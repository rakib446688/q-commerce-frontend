Q-Commerce Frontend - Folder and File Descriptions

This document describes each folder and file currently present in the repository.
It is intended to help explain the project structure and what each part does.

ROOT

- package.json
  Project manifest. Lists dependencies, scripts (dev/build), and metadata.

- package-lock.json
  Exact dependency lockfile generated by npm.

- vite.config.js
  Vite configuration for React build tooling.

- index.html
  HTML entry used by Vite to mount the React app.

- README.md
  Project documentation and setup notes.

- .gitignore
  Git ignore rules.

- .env
  Local environment variables (Supabase URL and key). Keep private.

- .env.example
  Example environment variable template (safe to share).

PUBLIC

- public/
  Static assets served at the root.

  - logo.jpg
    Brand logo used in the header.

SRC

- src/main.jsx
  Application entry. Mounts React and imports global styles and theme.

- src/App.jsx
  Root router and auth gate. Defines routes for Home, Shop, CategoryListing,
  Cart, Checkout, Account, Wishlist, Trending, Login, and Signup.

- src/Layout.jsx
  Top-level layout wrapper. Renders the top bar, search row, drawer, and footer.
  Handles search input routing to Shop with query.

- src/styles.css
  Global UI styles (layout, components, hero, tiles, rails, cards, footer, etc.).

- src/styles/
  - theme.css
    Light theme variables and utility classes.

COMPONENTS

- src/components/CategoryTiles.jsx
  Renders the five category image tiles (Women, Men, Kids, Electronics, Home & Living)
  with links to CategoryListing.

- src/components/ProductCard.jsx
  Product card UI: image, badges, brand, name, price, rating, swatches, actions.
  Uses product data provided by the store.

- src/components/ProductRail.jsx
  Horizontal product rail with scroll buttons and view-all link.

- src/components/QuickViewModal.jsx
  Quick view modal with color and size selection, add to cart, and wishlist support.

- src/components/Footer.jsx
  Black multi-column footer used on all pages.

CONTEXT

- src/context/AuthContext.jsx
  Supabase authentication context. Provides login, signup, logout, user state.

LIB

- src/lib/pages.js
  createPageUrl helper using pages.config.js (routing convention).

- src/pages.config.js
  Named routes mapping used by createPageUrl.

- src/lib/supabaseClient.js
  Supabase client configuration with env vars.

- src/lib/store/
  Local storage backed data modules.

  - productsStore.js
    Curated product list + helpers to fetch by category/id.

  - cartStore.js
    Cart CRUD in localStorage.

  - wishlistStore.js
    Wishlist CRUD in localStorage.

  - ordersStore.js
    Orders CRUD in localStorage.

  - trendingStore.js
    Interaction metrics (views/cart/wishlist) and trending selection.

PAGES

- src/pages/Home.jsx
  Main landing page. Hero image, category tiles, assistant banner, product rails.

- src/pages/CategoryListing.jsx
  Category/subcategory listing. Color filter and product grid.

- src/pages/Shop.jsx
  Full catalog page with optional query filtering.

- src/pages/Trending.jsx
  Trending grid (based on local interaction metrics).

- src/pages/Cart.jsx
  Cart view with quantities and totals (GBP + fixed shipping).

- src/pages/Checkout.jsx
  Checkout form; creates order and clears cart (UI-only payment).

- src/pages/Account.jsx
  Account overview with profile, actions, and recent orders.

- src/pages/Wishlist.jsx
  Wishlist view + empty state.

- src/pages/Login.jsx
  Supabase login page.

- src/pages/Signup.jsx
  Supabase signup page.

NOTES

- dist/ is build output (regenerated by npm run build).
- node_modules/ contains dependencies (regenerated by npm install).

END
